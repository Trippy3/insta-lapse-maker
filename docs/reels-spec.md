# Instagram Reels 出力仕様

CLI / Web UI どちらも同一の FFmpeg パイプラインを使うため、出力品質は完全に一致します。

## 基本仕様

| 項目 | 値 |
|------|---|
| コンテナ | MP4 |
| 映像コーデック | H.264 (High Profile, Level 4.0) |
| 解像度 | 1080 × 1920 (9:16) |
| フレームレート | 30fps |
| 映像ビットレート | 5 Mbps (最大 8 Mbps) |
| 音声コーデック | AAC-LC |
| 音声ビットレート | 128 kbps |
| サンプルレート | 48 kHz / ステレオ |
| 色空間 | BT.709 / yuv420p |
| faststart | 有効 (ストリーミング再生対応) |

## セーフゾーン

Reels の画面には UI 要素が上下に重なります。主要な被写体は中央領域 **1080 × 1220px** に収めることを推奨します。

```
┌───────────────┐ ← 上端
│  [上部 UI]    │ 約 250px: プロフィール・メニュー
├───────────────┤
│               │
│  安全領域     │ 約 1220px: 絵画が見える領域
│  1080×1220    │
│               │
├───────────────┤
│  [下部 UI]    │ 約 450–650px: キャプション・いいね・コメント
└───────────────┘ ← 下端 (1920px)
```

## 対応画像フォーマット

| 拡張子 | 形式 |
|--------|------|
| `.jpg` / `.jpeg` | JPEG |
| `.png` | PNG |
| `.heic` / `.heif` | HEIC (iPhone 標準) |
| `.webp` | WebP |
| `.bmp` | BMP |
| `.tiff` / `.tif` | TIFF |

## アスペクト比の調整モード

横長・縦長・正方形など異なるサイズの写真が混在しても自動調整します。

### `pad` (デフォルト)

元のアスペクト比を保ったまま、余白を黒帯で埋めます。

```
┌─────────────────┐
│ [黒帯]          │
├─────────────────┤
│                 │
│   元の画像      │
│                 │
├─────────────────┤
│ [黒帯]          │
└─────────────────┘
```

### `crop`

中央を基準にクロップし、黒帯なしで 9:16 を埋めます。

## プロキシ動画 (Web UI)

Web UI の編集中プレビュー用には軽量版を生成します。

| 項目 | 値 |
|------|---|
| 解像度 | 540 × 960 |
| フレームレート | 15fps |
| 用途 | ブラウザ確認専用 (Reels 投稿不可) |
